# restaurant-page
Pagina Web para Restaurante realizada con JavaScript


## Sobre el proyecto

El siguiente proyecto esta realizado como parte del curso de The Odin Project (NO es un proyecto guiado).
La intencion del mismo es usar Webpack como build tool y armar una pagina web creando todo el HTML mediante la modificacion del DOM en JavaScript.

Sin embargo, yo decidi usar Vite como build tool.
Tambien, al final, decidi agregar Lit para conocer su uso.
